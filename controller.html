模块化

  • 1.写模块 angular.module(name, array(依赖于谁))
  • 2.用模块 ng-app="module name"

    Controller

  • 核心、大段的代码要放入其中
  • Controller 是属于module的
  • 1. 写 
    •                     angular.module(name, [依赖])
                    module.controller(name, function() {
                        //Code
                        ...
                    })
  • 1. 用 
    •                     ng-app="module name"
                    ng-controller="controller name"

ng指令与JS不互通 打通指令与JS $scope作用域，一切数据都是由scope带来的

ng-controller 要先于ng-init执行

{{parseInt(a) + parseInt(b)}} 
            // ng-init="a=0;b=0"
            //1. Write Module
            let oModule = angular.module('test', []);
            //2. 写controller
            oModule.controller('ctrl1', function ($scope) {
                $scope.a = 0;
                $scope.b = 0;
                //代码
                $scope.parseInt = function(str) {
                    return parseInt(str);
                }
            });

$scope --- angular的作用域(所有的里面的东西，都在$scope上)
